# 🎥 Video Conferencing Platform

A modern full-stack video conferencing platform built with the MERN stack, Socket.IO, and WebRTC.

## 🚀 Features (Planned)

- User Authentication
- Create & Join Meetings
- Real-time Video & Audio Calls
- Screen Sharing
- In-meeting Chat
- Meeting Recording
- File Sharing
- Responsive UI
- Secure JWT Authentication

## 🛠️ Tech Stack

### Frontend
- React
- Tailwind CSS
- Socket.IO Client

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Socket.IO

### Real-time Communication
- WebRTC
- Socket.IO
